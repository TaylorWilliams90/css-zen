Purpose/Theme

We have chosen a theme of minimalism, inspired by the pages on Appleâ€™s website. We want to separate the content into small, impactful, digestible pieces for the user. The result will be a website with multiple sections that the user will scroll through. This will also enable us to make the website mobile responsive. We will add some kind of imagery in each of these segments. It would also be effective to add some subtle animations as the user scrolls through the content. Overall, we want the look and feel to be clean, readable, and modern.
